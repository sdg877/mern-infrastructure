A starter template for building full-stack applications using the MERN stack (MongoDB, Express, React, Node.js).

Installation
Clone the repository and navigate to the project directory.

Install backend dependencies by navigating to the backend folder and running npm install.

Install frontend dependencies by navigating to the frontend folder and running npm install.

Create a .env file in the backend directory with the following variables:

DATABASE_URL: Your MongoDB connection string (e.g., mongodb://localhost:27017/your-database-name).
VITE_BACKEND_URL: Backend URL (e.g., http://localhost:5000).
SECRET: Your JWT secret key.
Usage
Start the backend server by navigating to the backend folder and running npm run dev.

Start the frontend server by navigating to the frontend folder and running npm run dev.
